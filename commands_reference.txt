QF Interview Tool - Important Terminal Commands
==========================================

Backend (Python FastAPI)
-----------------------

# Start the backend server (from backend/python/src directory)
cd backend/python/src && python3 -m uvicorn main:app --reload --host 0.0.0.0 --port 8000
# This command:
# - Starts the FastAPI server on port 8000
# - Enables auto-reload for development
# - Allows connections from any IP (0.0.0.0)
# - Uses the 'app' variable from main.py

# Create and activate Python virtual environment
python3 -m venv venv
source venv/bin/activate  # On macOS/Linux
.\venv\Scripts\activate   # On Windows

# Install Python dependencies
cd backend/python && pip install -r requirements.txt
# or install individual packages:
pip install fastapi uvicorn python-dotenv numpy pandas scipy scikit-learn statsmodels yfinance

Frontend (React/Vite)
--------------------

# Start frontend development server (from frontend directory)
cd frontend && npm run dev
# Starts Vite dev server with hot reload

# Install frontend dependencies
cd frontend && npm install
# Install specific packages:
npm install @mui/material @emotion/react @emotion/styled axios react-router-dom

# Build frontend for production
cd frontend && npm run build
# Creates optimized build in dist directory

# Deploy to GitHub Pages
cd frontend && npm run deploy
# Builds and deploys to gh-pages branch

Git Commands
-----------

# Initialize repository
git init

# Add and commit changes
git add .
git commit -m "Initial commit: Project setup"

# Push to GitHub
git push origin main

# Deploy specific
git push -f origin main  # Force push (use carefully)
git subtree push --prefix frontend origin gh-pages  # Alternative way to deploy to GitHub Pages

# Create and switch to feature branch
git checkout -b feature/branch-name

# Push changes to remote
git push origin branch-name

# Merge changes
git checkout main && git merge feature/branch-name

# Utility Commands
# Kill process using specific port (e.g., 8000)
lsof -i :8000 | grep LISTEN | awk '{print $2}' | xargs kill -9
